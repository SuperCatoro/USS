# Layer 2 — Declarative Policy Engine

This layer transforms Structured Facts into policy decisions using deterministic declarative rules.

Key components:
- Datalog-based rule engine.
- Priority Ladder: Human Safety > Human Rights.
- Age-Stage Model:
  - Child (0–12)
  - Teen (13–19)
  - Self-Selecting Adults
  - Mid Career
  - Senior Career
- Versioned, transparent, and auditable logic.

This ensures governance cannot be influenced by machine learning outputs.
